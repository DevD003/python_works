 Literary & MorseCode-Wise Palindrome Check 
=======
**About:**


 A Simple Python Program for Checking the palindrome charecteristics of any ```Given String```

 * The Check for palindrome is done in :  
    * ```Literal``` check for ```word-wise palindrome``` 
    * ```MorseCode-wise``` check for ```symbol-wise palindrome```
